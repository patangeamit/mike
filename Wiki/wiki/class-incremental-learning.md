# Class-Incremental Learning

**Summary**: Class-incremental learning is a continual learning setting where models must handle new classes over time while retaining knowledge of previously seen classes. The source presents it as one of the main variants of continual learning. 

**Sources**: `raw/Continual Learning in Machine Learning.md`

**Last updated**: 2026-04-19

---

The source defines class-incremental learning as learning new classes or categories over time while maintaining knowledge of classes seen earlier (source: Continual Learning in Machine Learning.md). It gives image recognition as a representative use case, where new object classes may be introduced periodically (source: Continual Learning in Machine Learning.md).
