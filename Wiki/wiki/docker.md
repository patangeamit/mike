# Docker

**Summary**: Docker is an open-source containerization platform used to package applications and dependencies into portable containers. The DevOps roadmap presents it as a core tool for consistency, portability, and collaborative delivery.

The source describes Docker as an open-source containerization platform that packages software applications and their dependencies into small containers (source: Complete DevOps Roadmap - Beginner to Advanced.md). It says Docker is integral to the DevOps ecosystem because it provides portability, flexibility, scalability, efficient resource use, collaborative development, and consistency (source: Complete DevOps Roadmap - Beginner to Advanced.md).
