# Kubernetes

**Summary**: Kubernetes is an open-source platform for automating the deployment and management of large applications. The DevOps roadmap treats it as a major orchestration layer for scalable, automated operations. 

**Sources**: `raw/Complete DevOps Roadmap - Beginner to Advanced.md`

**Last updated**: 2026-04-19

The source describes Kubernetes, or K8s, as an open-source platform that automates deployment and management of large applications (source: Complete DevOps Roadmap - Beginner to Advanced.md). It says Kubernetes supports scalability and end-to-end process automation and associates it with the infrastructure-as-code principle for managing infrastructure resources (source: Complete DevOps Roadmap - Beginner to Advanced.md).

## Topics Mentioned

The article lists Kubernetes architecture, container orchestration, namespaces, images, jobs, labels and selectors, nodes, services, pods, replication controllers, ReplicaSets, deployments, volumes, and secrets as key topics (source: Complete DevOps Roadmap - Beginner to Advanced.md). These are presented as the main concepts a DevOps learner should understand once they already know containers and cloud platforms (source: Complete DevOps Roadmap - Beginner to Advanced.md).
