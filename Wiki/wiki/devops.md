# DevOps

**Summary**: DevOps is a set of practices that combines software development and IT operations to improve collaboration, automation, delivery speed, and operational reliability. In this wiki, it is treated as adjacent to but distinct from MLOps. 

**Sources**: `raw/Complete DevOps Roadmap - Beginner to Advanced.md`

**Last updated**: 2026-04-19

---

The source defines DevOps as a set of practices that combines software development and IT operations to support efficient and high-quality software delivery (source: Complete DevOps Roadmap - Beginner to Advanced.md). It emphasizes communication, collaboration, and integration between development and operations teams as central to the practice (source: Complete DevOps Roadmap - Beginner to Advanced.md).

## Main Responsibilities

The article says a DevOps engineer may implement CI/CD, automate infrastructure, work across development and operations teams, manage application availability and performance, and troubleshoot operational issues (source: Complete DevOps Roadmap - Beginner to Advanced.md). These responsibilities reflect the article's view of DevOps as both an engineering practice and an operational role (source: Complete DevOps Roadmap - Beginner to Advanced.md).

## Core Skill Areas

The roadmap ties DevOps to programming, operating systems, networking, Linux, source control, cloud computing, Docker, Kubernetes, and supporting tools such as Terraform, Prometheus, and Grafana (source: Complete DevOps Roadmap - Beginner to Advanced.md). The DevOps lifecycle is described in the source as a continuous process that includes planning, building, integrating, deploying, monitoring, operating, and continuous feedback (source: Complete DevOps Roadmap - Beginner to Advanced.md).

## Relationship to MLOps

The wiki should treat [[devops]] and [[mlops]] as related but separate pages (inference: this is a wiki modeling decision based on both sources). The MLOps source explicitly says MLOps unifies machine learning, DevOps, and data engineering, which implies that DevOps is a foundation or neighboring discipline rather than the same thing (source: Complete DevOps Roadmap - Beginner to Advanced.md; source: The Ultimate MLOps Roadmap for 2026 A Complete Guide to Becoming an MLOps Engineer.md).
