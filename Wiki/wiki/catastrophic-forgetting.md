# Catastrophic Forgetting

**Summary**: Catastrophic forgetting is the loss of performance on previously learned tasks when a model learns new ones. The continual learning source treats it as the core problem that continual learning methods try to mitigate. 

**Sources**: `raw/Continual Learning in Machine Learning.md`

**Last updated**: 2026-04-19

---

The source describes catastrophic forgetting as the tendency of traditional models to lose proficiency in previously learned tasks when exposed to new ones (source: Continual Learning in Machine Learning.md). It presents this as the main reason continual learning is needed in dynamic environments (source: Continual Learning in Machine Learning.md).

## Mitigation Approaches

The article names regularization methods such as Elastic Weight Consolidation and Synaptic Intelligence as strategies for stabilizing important weights while learning new tasks (source: Continual Learning in Machine Learning.md). It also points to replay mechanisms, memory systems, knowledge distillation, and transfer learning as ways to preserve or reinforce old knowledge during further learning (source: Continual Learning in Machine Learning.md).
